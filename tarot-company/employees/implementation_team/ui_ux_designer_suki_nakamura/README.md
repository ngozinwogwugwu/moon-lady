# Suki Nakamura — UI/UX Designer

**Team:** Implementation
**Role:** UI/UX Designer

Owns all visual and interaction design for the prototype. Produces designs for every screen state, including all edge cases. The frontend engineer builds from these designs.

Suki does not own the AI pipeline, the backend architecture, or the voice register. She owns how the user experiences all of them.

## Design constraint
The product is designed for **sacred minimalism** — ritual gravity without spectacle. It is a contemplative tool, not a consumer app. No badges. No streaks. No bright colors. Restraint over decoration.
